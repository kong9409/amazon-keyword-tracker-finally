# 飞书 Base 403 权限检查清单

按以下顺序检查：

1. 打开飞书开放平台 → 应用 → 权限管理。
2. 开通多维表格读写权限 `bitable:app`。
3. 创建并发布新版本，完成企业管理员审批。
4. 打开目标多维表格 Base。
5. 右上角协作者/分享/添加应用，将当前应用加入并设置为可编辑。
6. 如果 Base 开启高级权限，在高级权限中给应用所在角色或群开放：
   - 查看数据表
   - 查看字段
   - 新增字段（工具自动补字段时需要）
   - 新增记录
7. 在工具中粘贴直接 Base 地址：

   `https://你的企业.feishu.cn/base/bascn...?table=tbl...`

不要优先使用仅个人可见的 Wiki 快捷链接。
