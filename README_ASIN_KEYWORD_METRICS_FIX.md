# ASIN 维度关键词指标修复说明

本版本修正了关键词指标和产品指标的归属逻辑：

1. `ABA热度排名`、`搜索量`、`关键词流量占比` 只从该 ASIN 的 `product_traffic_terms` 匹配行读取。
2. 不再把泛关键词搜索结果里的流量占比/ABA/搜索量填到 ASIN 行里，避免出现数据错位。
3. 排序改为按 ASIN 分组后排序：每个 ASIN 下面监控的关键词，按该 ASIN 对应关键词的 `关键词流量占比` 降序排列。
4. `价格`、`优惠券类型`、`优惠券优惠`、`是否秒杀`、`秒杀价格`、`Prime专享价/折扣价`、`月销量`、`大类排名`、`评分`、`评价数` 都从该 ASIN 的产品详情/趋势数据读取，并按 ASIN 缓存，避免跨 ASIN 混用。
5. Excel 全部左对齐，并取消自动换行。

如果 Sorftime 没有返回某个字段，会保留空白，不会用其它 ASIN 或泛关键词数据代替。
