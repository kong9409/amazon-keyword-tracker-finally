# V7 修复说明 · 字段前置与西柚接口校正

## 1. 监控字段前置

页面新增 STEP 1“监控字段”，先固定展示 18 个结果字段。切换 Sorftime、卖家精灵、SIF、西柚洞察或其他 MCP/API 时，会基于 `static/field-mapping.json` 显示每个字段对应的接口和覆盖状态。

映射依据为用户上传的《各插件 MCP 目录表.xlsx》。SIF 工作表没有具体工具目录，因此使用实时 `tools/list`、工具说明和 `inputSchema` 动态匹配。

## 2. 流量占比

统一输出百分比并保留两位小数：

- `0.125` → `12.50%`
- `12.5` → `12.50%`
- Excel 写入真实数值并应用 `0.00%` 格式。

## 3. 新增小类排名

在“大类排名”后增加“小类排名”，同步覆盖：

- 页面结果表
- Excel 数据表
- 飞书字段
- SQLite 数据表自动迁移
- 多数据源字段解析

小类排名属于条件字段；接口仅返回根类目时保持空白，不影响其他字段状态。

## 4. 西柚洞察

- 广告位：优先调用 `get_asin_keyword_rank_trends` 并识别广告排名；工具 schema 支持排名类型时会补发广告类型请求。
- 月销量：调用 `get_asin_order_trends`，在返回的月份序列中选择当前自然月。
- 大类排名：`get_asin_bsr_trends` 的根类目。
- 小类排名：`get_asin_bsr_trends` 中最深的非根类目。
